# DigiTools - Premium Digital Tools Marketplace

A modern, fully responsive e-commerce platform for digital tools and premium assets. Built with React and Tailwind CSS.

## 🌟 Live Demo

[View Live Demo](https://digital-tools-website-0097.netlify.app/)

## 📖 Description

DigiTools is a comprehensive digital marketplace where users can browse, select, and purchase premium digital tools including AI writing assistants, design templates, stock assets, automation toolkits, and more. The platform offers a seamless shopping experience with real-time cart updates, toast notifications, and responsive design.

## 🛠️ Technologies Used

- **React.js** - Frontend framework
- **Tailwind CSS** - Utility-first CSS framework
- **DaisyUI** - Component library for Tailwind
- **React Toastify** - Toast notifications
- **React Icons** - Icon library
- **LocalStorage** - Cart persistence
- **JSON** - Product data management

## ✨ Features

1. **Smart Cart Management**
   - Add/remove products with quantity controls
   - Real-time cart total calculation
   - Persistent cart storage using localStorage
   - Toast notifications for all cart actions

2. **Interactive Product Display**
   - 3-column responsive product grid
   - Product tags (Popular, New, Best Seller)
   - Feature lists with checkmarks
   - Buy now button with visual feedback

3. **Modern UI/UX**
   - Fully responsive design (mobile to desktop)
   - Smooth animations and transitions
   - Dark/light theme ready
   - Tabbed interface for Products/Cart views
